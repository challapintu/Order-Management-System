# Order Management System

A full-stack Order Management System built using **Spring Boot** for the backend and **Angular** for the frontend.  
The application is containerized using **Docker** and deployed on **Microsoft Azure**.

---

## Tech Stack

### Backend
- Java
- Spring Boot
- Spring Security (JWT)
- JPA / Hibernate
- MySQL

### Frontend
- Angular
- TypeScript
- HTML, CSS

### DevOps / Cloud
- Docker
- Microsoft Azure
