# Order Management System

Order Management System built using Spring Boot and Angular.

## Tech Stack
- Spring Boot
- Angular
- MySQL
- REST APIs

## Project Structure
- backend/ : Spring Boot application
- frontend/ : Angular application

## Features
- User Authentication
- Product Management
- Order Placement
- Order Tracking

## How to Run
### Backend
- Import backend into Eclipse/IntelliJ
- Update DB config in application.properties
- Run as Spring Boot App

### Frontend
- Open frontend in VS Code
- Run `npm install`
- Run `ng serve`
